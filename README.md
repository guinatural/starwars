## Como Executar Localmente

To run this project locally, follow these comprehensive setup instructions:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/guinatural/starwars.git
   cd starwars
   ```

2. **Install Dependencies**
   Ensure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed. Then run:
   ```bash
   npm install
   ```

3. **Set Up Environment Variables**
   Create a `.env` file in the root directory and update it with the necessary configurations.

4. **Run the Application**
   Use the following command to start the application:
   ```bash
   npm start
   ```

5. **Accessing the Application**
   Open your browser and navigate to `http://localhost:3000` to view the application.
